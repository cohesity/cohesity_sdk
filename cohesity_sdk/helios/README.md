## Notes for Large OpenAPI documents
If the OpenAPI document is large, imports in cohesity_sdk.apis and cohesity_sdk.models may fail with a
RecursionError indicating the maximum recursion limit has been exceeded. In that case, there are a couple of solutions:

Solution 1:
Use specific imports for apis and models like:
- `from cohesity_sdk.api.default_api import DefaultApi`
- `from cohesity_sdk.model.pet import Pet`

Solution 2:
Before importing the package, adjust the maximum recursion limit as shown below:
```
import sys
sys.setrecursionlimit(1500)
import cohesity_sdk
from cohesity_sdk.apis import *
from cohesity_sdk.models import *
```
